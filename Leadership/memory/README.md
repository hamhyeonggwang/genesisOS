# Leadership Domain Memory

팀 리더십 업무에서 배운 패턴, 피드백, 결정 규칙을 기록합니다.

## 파일 구조

- `leadership-feedback.md` — 팀 관리, 기획에서 배운 것들
- `team-patterns.md` — 팀원별 특성, 성과 패턴
- `decision-log.md` — 중요 의사결정 및 이유

## 예시

```markdown
# 팀 리더십 피드백

## 팀 조정
- 주간 회의 30분: 조정 효율 최적
- 비동기 소통: 문서 기반 → 이해도 ↑

## 팀원 관리
- 목표 명확화: 성과 평가 정확성 ↑
- 피드백 타이밍: 주 1회 일대일 → 신뢰도 ↑

## 의사결정
- 팀 의견 수렴: 48시간 협의 → 합의도 ↑
```
