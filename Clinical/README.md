# Clinical OT Domain

작업치료 임상 실무 영역입니다.

## 구조

- **projects/** — 클라이언트별 평가, 중재, 결과 관리
- **.claude/** — 임상 도메인 설정 (settings.json)
- **memory/** — 임상 작업 피드백 및 규칙

## 빠른 시작

- `Ctrl+R` — `/report` 임상 보고서 자동화
- 클라이언트 폴더 생성: `projects/[client-name]/`
  - evaluation/ — 초기평가, 추적평가
  - intervention/ — 중재 계획, 회기 기록
  - outcome/ — 결과 요약, 퇴원 보고

## 참고

- ICF 매핑: `Shared/skills/icf-mapper/`
- 평가도구: `Shared/knowledge/assessment-tools/`
- OT 표준용어: `Shared/knowledge/ot-standards/`
