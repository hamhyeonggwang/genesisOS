# Design Prototyper Skill

**UI/UX 디자인 → 프로토타입**을 빠르게 구축하는 스킬입니다.

## 목적

- 개념 → Figma 와이어프레임 변환
- 목업 → 상호작용 프로토타입
- 사용자 흐름 시각화
- 개발 전 디자인 검증

## 사용 흐름

1. **요구사항 분석** — 사용자 니즈, 기능 정의
2. **와이어프레임** — 저충실도 레이아웃 설계
3. **목업** — 고충실도 비주얼 디자인 (Figma)
4. **프로토타입** — 상호작용 추가, 사용자 테스트

## 주요 기능

- Figma 자동 프로젝트 생성
- 와이어프레임 템플릿 활용
- 디자인 시스템 기반 목업
- 사용자 흐름 다이어그램
- 인터랙션 프로토타입

## 작업 흐름

```
Design/projects/[project-name]/
├─ wireframes/         (저충실도 스케치)
├─ mockups/           (Figma 고충실도 목업)
├─ design-system/     (색상, 컴포넌트, 토큰)
└─ assets/            (이미지, 아이콘)
```

## 참고

- Figma MCP: 자동 활성화 (Design 폴더에서)
- 디자인 시스템: `Design/design-tokens/`
- UX 검토: `Shared/skills/ux-reviewer/`
- 개발 연동: `Development/projects/`
