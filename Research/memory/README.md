# Research Domain Memory

연구 작업 중 배운 패턴, 근거검토 기준, 분석 팁을 기록합니다.

## 파일 구조

- `research-feedback.md` — 연구 방법론, 문헌검토 팁
- `quality-appraisal-notes.md` — 질 평가 기준 및 경험
- `methodology-notes.md` — 연구설계별 주의사항

## 예시

```markdown
# 연구 피드백

## 문헌검토
- RCT 논문: Cochrane Risk of Bias 2 도구 사용 ↓ 시간
- 체계적고찰: PRISMA 체크리스트 필수

## 질 평가
- 샘플 크기 <30: 주의깊게 해석
- Allocation concealment 명시 안 됨 = 높은 편향 위험
```
