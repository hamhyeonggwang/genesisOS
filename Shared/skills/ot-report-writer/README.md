# OT Report Writer Skill

작업치료 평가 및 중재 보고서를 **자동 생성**하는 스킬입니다.

## 목적

- 임상 기록 → 전문 보고서 변환
- 평가 소견 체계적 문서화
- 퇴원/전원 요약 자동화

## 사용 흐름

1. **평가 원데이터 입력** — 클라이언트 정보, 검사 결과
2. **보고서 자동화** — 구조화된 평가 보고서 생성
3. **중재 기록 변환** — 회기 기록 → 중재 효과 요약
4. **퇴원 요약** — 치료 과정, 결과, 권고사항

## 주요 기능

- 초기평가 보고서 자동 작성
- ICF 기반 목표 → 중재 전략 매핑
- 회기별 기록 → 주간/월간 진행 요약
- 퇴원 보고서 (결과 평가, 권고사항)

## 보고서 템플릿

```
1. 클라이언트 정보
2. 평가 검사 결과
3. ICF 기반 문제 분석
4. 작업치료 목표
5. 중재 전략
6. 예상 결과
```
