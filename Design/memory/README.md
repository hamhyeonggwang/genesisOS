# Design Domain Memory

디자인 작업에서 배운 패턴, 프로토타이핑 팁을 기록합니다.

## 파일 구조

- `design-feedback.md` — 디자인 원칙, 효율화 팁
- `component-patterns.md` — 재사용 컴포넌트 패턴
- `design-system-notes.md` — 디자인 시스템 관리

## 예시

```markdown
# 디자인 피드백

## 와이어프레임
- 충실도 낮음: 빠른 반복 ↑
- 화면 흐름: 사용자 시나리오 기반 → 이해도 ↑

## 목업
- Figma 컴포넌트: 일관성 ↑, 수정 시간 ↓
- 디자인 토큰: 색상/타입 관리 체계적

## 프로토타입
- 인터랙션 검증: 개발 전 우선
```
