# Research Domain

연구 및 근거 기반 실무 영역입니다.

## 구조

- **papers/** — 저술 중인 논문, 발표된 논문
- **literature/** — 문헌 검토, 질 평가, 주제별 정리
- **datasets/** — 원본 데이터, 처리된 데이터, 분석 결과
- **.claude/** — 연구 도메인 설정 (settings.json)
- **memory/** — 연구 작업 피드백 및 규칙

## 빠른 시작

- `Ctrl+P` — `/paper` 논문 작성 자동화
- literature/ — 주제별 폴더 생성
  - papers.md — 논문 목록
  - critical-appraisal/ — 질 평가 문서
  - summary.md — 요약 및 결론
