# Design Domain

UI/UX 디자인, 웹페이지, 프로토타이핑을 관리합니다.

## 구조

- **projects/** — 디자인 프로젝트 (와이어프레임, 목업, 디자인 시스템)
- **components/** — 재사용 UI 컴포넌트
- **design-tokens/** — 색상, 타이포그래피, 간격 등
- **.claude/** — 디자인 설정 (Figma MCP, Ctrl+I: /design)
- **memory/** — 디자인 작업 피드백

## 빠른 시작

- `Ctrl+I` — `/design` 디자인→프로토타입 자동화
- 프로젝트: `projects/[project-name]/`
  - wireframes/ — 초기 와이어프레임
  - mockups/ — 고충실도 목업
  - design-system/ — Figma 디자인 시스템
  - assets/ — 이미지, 아이콘

## 참고

- 디자인 프로토타이핑: `Shared/skills/design-prototyper/`
- Figma 연동: MCP 도구 자동 활성화
- UX 검토: `Shared/skills/ux-reviewer/`
