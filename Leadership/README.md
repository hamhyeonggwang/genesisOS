# Leadership Domain

팀 리더십, 기획, 조정 업무를 관리합니다.

## 구조

- **team/** — 팀 관리 (팀원, 프로젝트, 계획, 회의)
- **reporting/** — 주간/월간/전략 보고서
- **.claude/** — 리더십 설정 (settings.json, Ctrl+T: /team-report)
- **memory/** — 리더십 업무 피드백 및 규칙

## 빠른 시작

- `Ctrl+T` — `/team-report` 팀 주간/월간 보고서 자동화
- 팀 프로젝트: `team/projects/`에서 관리
- 팀 계획: `team/planning/` (전략 문서, OKR, 타임라인)
- 회의록: `team/meetings/` (의사결정 로그)

## 참고

- 팀 보고서 스킬: `Shared/skills/team-report-writer/`
- 프로젝트 관리: `Shared/skills/project-manager/`
- 템플릿: `Shared/templates/`
