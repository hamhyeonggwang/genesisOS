# Clinical Domain Memory

임상 작업 중 배운 패턴, 피드백, 규칙을 기록합니다.

## 파일 구조

- `clinical-feedback.md` — 임상 작업에서 배운 것들
- `client-patterns.md` — 클라이언트별 특성, 경향
- `intervention-notes.md` — 중재 효과적인 기법들

## 예시

```markdown
# 임상 피드백

## 평가 관련
- 감각통합 선호도 높은 클라이언트: SIPT 후 중재 효과 ↑

## 중재 관련
- 한 회기에 3가지 이상 활동: 피로도 증가, 참여도 ↓
- 루틴 화: 첫 5분 열신운동 → 참여도 안정적

## 보고서 작성
- ICF 목표 설정: 클라이언트 관점 명시 필수
```
