# Team Report Writer Skill

**팀 주간/월간 보고서**를 자동 생성하는 스킬입니다.

## 목적

- 팀 진행도 → 구조화된 보고서 변환
- 팀원 성과 요약
- 의사결정 로그 자동화
- 리더십 효율화

## 사용 흐름

1. **팀 데이터 수집** — 팀원 진행도, 완료 항목, 이슈
2. **보고서 자동화** — 주간/월간 구조화된 보고서 생성
3. **의사결정 기록** — 중요 결정사항, 이유 요약
4. **경영진 보고** — 임원진 대상 전략 보고서

## 주요 기능

- 주간 팀 보고서 (팀원별 진행도, KPI, 이슈)
- 월간 팀 보고서 (달성도, 개선사항, 다음달 계획)
- 전략 보고서 (분기별 성과, 목표 달성도)

## 보고서 템플릿

```
1. 팀 개요
2. 주간 성과 요약
3. 팀원별 진행도
4. 주요 이슈 및 해결 방안
5. KPI 진행률
6. 다음주 계획
7. 의사결정 로그
```
