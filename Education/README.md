# Education Domain

강의 및 교육 콘텐츠 개발 영역입니다.

## 구조

- **lectures/** — 강의별 계획서, 강의안, 평가
- **slides/** — 강의 슬라이드 (PPTX, PDF)
- **materials/** — 강의 보조자료, 실습 자료
- **.claude/** — 교육 도메인 설정 (settings.json)
- **memory/** — 교육 작업 피드백 및 규칙

## 빠른 시작

- `Ctrl+L` — `/lecture` 강의 준비 자동화
- lectures/[course-name]/ 생성:
  - syllabus.md — 강의계획서
  - learning-objectives.md — 학습목표
  - session-plans/ — 회차별 계획
  - assessments/ — 평가 문제

## 참고

- 강의 기획: `Shared/skills/lecture-planner/`
- 슬라이드 생성: `Shared/skills/slide-builder/`
- 퀴즈 생성: `Shared/skills/quiz-generator/`
