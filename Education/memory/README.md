# Education Domain Memory

교육 설계 및 강의 운영 중 배운 패턴을 기록합니다.

## 파일 구조

- `education-feedback.md` — 강의 설계, 학생 피드백 분석
- `lecture-tips.md` — 각 주제별 효과적인 설명 방법
- `assessment-notes.md` — 평가 설계 및 해석

## 예시

```markdown
# 교육 피드백

## 강의설계
- 1 슬라이드 = 1 메시지: 학생 이해도 ↑
- 임상 사례 2개 이상: 이론-실무 연결 명확

## 평가
- 선택형: 개념 이해도 측정
- 서술형: 임상 적용 능력 측정
```
